## 项目实践

### 商铺系统

* 技术栈：
  - 前端：前端UI库 SUI Mobile/jQuery/Chrome UA
  - 后端：SSM／Spring Boot／图片开源工具 Thumbnailator/
    验证码组件Kaptcha／作业调度框架 Quartz/Shell
  - 数据库：MySQL
  - 缓存：Redis
  - 部署：阿里云
  - 技术亮点：MySQL主从同步/项目读写分离/框架迁移/代码重构

### 接口自动化测试框架

* 技术栈：MUI+H5Plus、MariaDB/MySQL、SpringBoot+Netty、Nginx+FastDFS
